# 아가밥

아기 이유식 재료 재고, 유통기한, 맛/영양 조합을 고려한 식단 추천, 식단 편집,
식사 후 아기의 섭취량·반응 기록까지 관리하는 로컬 웹 앱.

## 화면 구성

- **식단 추천**: 재고 상태(유통기한 임박, 최근 사용 이력, 아기 반응, 영양 균형)를 반영해
  하루 식단(곡물/단백질/채소 조합)을 추천하고, 재료 추가·삭제·수량 조정 후 확정하면
  재고가 자동으로 차감됩니다.
- **재고**: 재료별 수량, 보관 위치, 유통기한을 관리하고 임박 항목을 강조 표시합니다.
- **재료**: 재료 이름, 카테고리, 시작 가능 이유식 단계, 맛/영양 태그, 알레르기 여부를 관리합니다.
- **기록**: 확정된 식단마다 아기가 먹은 양과 반응(좋아함/보통/싫어함/알레르기 의심)을 기록하며,
  이 기록은 다음 식단 추천 점수에 반영됩니다.

## 데이터 저장

모든 데이터는 브라우저 IndexedDB(Dexie)에 로컬로 저장됩니다. 별도 로그인/서버가 없으며,
같은 브라우저·기기에서만 데이터가 유지됩니다.
